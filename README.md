# ✈️ AeroIQ — AI-Powered Flight Search Assistant

🌍 **AeroIQ** is an AI-powered flight search assistant that helps users discover flights through natural language. Instead of filling out long forms and endless filters, users can simply describe where they want to go, when they want to travel, and what kind of option they prefer.

The project includes a premium landing page for the **AeroIQ Flight AI Agent**, a custom full-screen chat interface, 🤖 Telegram bot access, and ⚡ live webhook integration for real-time AI responses.

---

# 🚀 Overview

AeroIQ is designed to make flight search feel **fast, conversational, and modern**. Users can search flights naturally with prompts like:

- 💸 “Cheapest Mumbai to Dubai, 20 May, 1 adult”
- ⚡ “Fastest Delhi to Singapore, 15 May, 2 adults”
- 💎 “Best value London to Tokyo, 10 June, 1 adult”
- 🔁 “Mumbai to Goa round trip, 30 May return 2 June”

The app sends the user’s request to an AI workflow and delivers intelligent responses inside a sleek conversational interface.
